Semantic Tags in HTML

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of semantic elements: < form >, < table >, and < article > - Clearly defines its content.

5 Semantic Tags in HTML

  1. < section >: "A section is a thematic grouping of content, typically with a heading."
  2. < article >: An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.
  3. < header >: represents a part of a page which is meant to be introductory. It can include heading tags, a logo, a search bar, and navigation elements.
  4. < footer >: A footer for a document or section.
  5. < nav >: represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
