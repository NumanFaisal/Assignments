Tags, Element and Attributes

Tags, elements, and attributes are essential components of HTML, the markup language used to create web pages

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags. Some example of HTML tags are:

  1. < title > = Defines a title for the document.
  2. < body > = Defines the document's body.
  3. < h1 > to < h6 > = Defines HTML headings.

An HTML file is made of elements. These elements are responsible for creating web pages and define content in that webpage. An element in HTML usually consist of a start tag , close tag and content inserted between them. Technically, an element is a collection of start tag, attributes, end tag, content between them. Some exmaple of element

  1. "h" = Defines as Heading.
  2. "p" = Defines as Paragraph.
  3. "img" = Defines as Image.

Elements in HTML have attributes ; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want. Some exmaple of attribute.

  1. Accept = List of types the server accepts, typically a file type.
  2. Accesskey = Keyboard shortcut to activate or add focus to the element.
  3. Allow = Specifies a feature-policy for the iframe.
